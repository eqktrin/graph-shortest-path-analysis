# String Search & Puzzle Solver Lab

## Описание
Данный проект состоит из двух заданий, выполненных в рамках лабораторной работы по алгоритмам и структурам данных:

### Задание 1 — Поиск подстроки
Реализованы алгоритмы поиска подстроки в строке:
- **Алгоритм Кнута-Морриса-Пратта (KMP)**  
- **Упрощенный алгоритм Бойера-Мура**  

Функциональность:
- Ввод строки и подстроки с клавиатуры  
- Выбор чувствительности к регистру  
- Оценка времени выполнения каждого алгоритма  
- Сравнение с стандартной функцией поиска Python (`str.find()`)

Использованные модули:
- `datetime` — для замера времени выполнения  
- `collections.defaultdict` — для построения таблицы сдвигов в алгоритме Бойера-Мура  

---

### Задание 2 — Пятнашки (15-пазл)
Программа проверяет, **является ли данное расположение "решаемым"**, и если возможно, находит хотя бы одно решение:

- Входные данные: массив чисел 0–15 (0 обозначает пустое поле)  
- Выходные данные: последовательность перемещений или пустой массив, если решение невозможно  

Алгоритм:
- Проверка решаемости позиции  
- Поиск решения через **поиск в ширину (BFS)**  
- Используются структуры данных: `deque`, `set`
